# MASOFISH Fish Image Sources

The Fish ID species cards use representative reference images hosted by Wikimedia Commons.
Use the **PHOTO SOURCE** link on each card to view the original file page, author, and license.

These pictures are visual references for the model class and are not the images used to train the Teachable Machine model.
