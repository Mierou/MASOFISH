Supabase Setup Required

Connect the login system

The login and sign-up interface is installed, but the Supabase project URL and public key still need to be added.

  1. Create or open a Supabase project.
  2. Open Project Settings → API.
  3. Copy the Project URL and Publishable/anon key.
  4. Paste them into supabase-config.js.
  5. Add the Vercel domain to the Supabase Auth URL settings.
Never put the Supabase service_role key in this browser application.

Prototype mode skips real authentication and is only intended for testing the interface.
