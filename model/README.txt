MASOFISH Fish Identification Model

This package contains a valid TensorFlow.js model exported from Teachable Machine.

Files:
- model.json
- metadata.json
- weights.bin

Classes:
1. Milkfish
2. Indian Mackerel
3. Eel
4. Tilapia
5. Scatfish
6. Mullet
7. Rabbitfish
8. No Fish

Note:
The spelling in metadata.json was corrected from "Indian Mackarel" to "Indian Mackerel".
The model's class order was not changed.
